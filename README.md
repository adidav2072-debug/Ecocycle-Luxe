## Project Description
EcoCycle Luxe is a professional digital portfolio showcasing high‑end sculpture‑like art created from recycled electronic waste (e‑waste). The project demonstrates how discarded industrial materials can be transformed into luxury décor, combining sustainability with modern aesthetic design.  

The updated Part 2 version of the website presents a structured five‑page navigation system, refined styling, and a clear communication channel for inquiries and contact.

## Features
- **Home Page:** Introduces EcoCycle Luxe with an overview of the organisation, its mission, and a tagline *“Reinventing E‑Waste into Modern Décor”*.  
- **Process Page:** Explains the three stages of craftsmanship – sourcing, component refinement, and sculpting & fusion – with supporting visuals.  
- **Gallery Page:** Displays signature e‑waste sculptures including *The Synthetic Soul*, *Electric Afterlife*, *Circuit Fragment*, and *The Iron Glaze*.  
- **Inquiry Page:** Provides a secure form for potential clients to request custom designs or express interest in specific pieces.  
- **Contact Page:** Lists business hours, email, phone, and physical address in Polokwane, South Africa, alongside social media links.
