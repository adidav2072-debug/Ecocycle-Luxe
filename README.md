# Ecocycle-Luxe

## Project Description
Ecocycle-Luxe is a web-based portfolio showcasing high-end sculptural art created from recycled electronic waste. This project aims to highlight the intersection of sustainability and luxury design.

## Features
* **Gallery:** High-quality images of e-waste sculptures.
* **Process:** A step-by-step look at how materials are transformed.
* **Inquiry:** A contact form for custom commissions.
* **Sitemap:** Easy navigation of the site structure.

## Tech Stack
* HTML5
* Git/GitHub
* Font Awesome (Icons)
