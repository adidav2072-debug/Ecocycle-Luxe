# Ecocycle-Luxe Digital Portfolio
## Project Description
Ecocycle-Luxe is a professional digital portfolio showcasing high-end sculpture-like art created from recycled electronic waste (e-waste). The project shows how industrial waste can be transformed into luxury décor, combining environmental sustainability with modern aesthetic design.
## Features
 * **Gallery:** Displays signature e-waste sculptures such as "The Synthetic Soul" and "Electric Afterlife".
 * **Process:** Explains the sourcing and refinement stages, focusing on industrial centers,essecially in Polokwane and all over South Africa.
 * **Inquiry:** Provides a secure contact method for any interest in the art for purchase.
 * **Sitemap:** Offers a clear engagement of the platform's five-page structure for efficient navigation.
